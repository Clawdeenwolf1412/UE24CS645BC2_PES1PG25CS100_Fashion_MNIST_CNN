# Fashion MNIST CNN from Scratch

## Objective
This project implements a Convolutional Neural Network (CNN) from scratch using NumPy for Fashion MNIST image classification.

## Features
- Convolution Layer
- Max Pooling Layer
- Fully Connected Layer
- Forward Pass
- Backward Pass
- Training and Testing
- Accuracy and Loss Graphs

## Dataset
Fashion MNIST Dataset

## Libraries Used
- NumPy
- Matplotlib
- TensorFlow

## How to Run
1. Open Jupyter Notebook
2. Run all cells sequentially

## Output
The CNN classifies Fashion MNIST clothing images into 10 categories.
